---
title: UsuarioService
description: Esta seção descreve o serviço de gerenciamento de usuários da aplicação, que é responsável por gerenciar as operações relacionadas aos usuários.
---

## Funções do Serviço

### save

```java
@Transactional
@Override
public void save(UsuarioRequestDto request)
```

Este método salva um novo usuário na aplicação.

### uploadAnexos

```java
@Transactional
@Override
public void uploadAnexos(String login, List<TipoDocumento> tipoDocumentos, List<MultipartFile> arquivos)
```

Este método faz o upload de anexos para um usuário.

### findById

```java
@Override
public Usuario findById(UUID id)
```

Este método busca um usuário pelo ID.

### findByLogin

```java
@Override
public Usuario findByLogin(String login)
```

Este método busca um usuário pelo login.

### findByLoginAtivo

```java
@Override
public Usuario findByLoginAtivo(String login)
```

Este método busca um usuário ativo pelo login.

### findByFirebaseUID

```java
@Override
public Usuario findByFirebaseUID(String uid)
```

Este método busca um usuário pelo UID do Firebase.

### validarCodigo

```java
@Transactional
@Override
public void validarCodigo(String login, String codigo)
```

Este método valida um código de verificação para um usuário.

### reenviarCodigo

```java
@Transactional
@Override

```

### delete

@Override
