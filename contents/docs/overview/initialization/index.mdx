# Raízes Cearenses API - Inicialização

## Pré-requisitos

Certifique-se de ter os seguintes softwares instalados no seu ambiente:

- **Docker** - Para gerenciar containers.
- **Docker Compose** - Para orquestrar os containers.
- **Maven** - Para compilar e executar a aplicação.

---

## Configuração Inicial

### 1. Configurar o arquivo `.env`

Antes de começar, você deve configurar as variáveis de ambiente necessárias para a aplicação.

1. Copie o arquivo `.env.example` para `.env`:
    ```bash
    cp .env.example .env
    ```

2. Edite o arquivo `.env` para preencher as variáveis de ambiente específicas do seu ambiente:
    ```bash
    nano .env
    ```

---
