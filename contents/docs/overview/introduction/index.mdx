A API "Raízes Cearenses" é uma solução poderosa e completa para o gerenciamento de usuários, autenticação, envio de mensagens e termos de uso. 🚀

Ela oferece uma ampla gama de funcionalidades, incluindo:

- Gestão de Usuários: Criação, atualização e exclusão de usuários de forma simples e eficiente. 👥
- Segurança: Mecanismos robustos de autenticação e autorização para proteger os dados e acessos. 🔒
- Comunicação Integrada: Suporte ao envio de mensagens via WhatsApp e e-mail, proporcionando uma comunicação ágil e eficaz com os usuários. 💬📧
- Políticas Atualizadas: Consulta e aceitação dos termos de uso mais recentes, garantindo que os usuários estejam sempre cientes das políticas vigentes. 📜✅
- Monitoramento de Saúde: Endpoints dedicados para verificar a saúde da API, assegurando alta disponibilidade e confiabilidade. ⚙️📶

Com essas funcionalidades, a API "Raízes Cearenses" se destaca como uma ferramenta essencial para quem busca eficiência, segurança e comunicação simplificada no gerenciamento de sistemas. 🌟



Aqui está uma versão revisada e aprimorada do texto com emojis para destacar as diferentes camadas e componentes da arquitetura:

A arquitetura deste projeto segue um padrão de camadas amplamente utilizado em aplicações Spring Boot, garantindo organização, escalabilidade e manutenibilidade. 🏗️✨ Aqui está um resumo das principais camadas e componentes:

🌐 Controllers
Responsáveis por receber as requisições HTTP, processar os dados de entrada e retornar respostas apropriadas.

⚙️ Services
Contêm a lógica de negócios da aplicação, processando os dados recebidos dos controllers e interagindo com os repositórios.

💾 Repositories
Gerenciam a interação com o banco de dados, utilizando Spring Data JPA para operações CRUD.

📋 Entities
Representam as tabelas do banco de dados e são mapeadas com a anotação @Entity.

📦 DTOs (Data Transfer Objects)
Facilitam a transferência de dados entre as camadas da aplicação, encapsulando informações para evitar a exposição direta das entidades.

🛠️ Helpers
Incluem métodos utilitários que auxiliam na validação e processamento de dados.

❗ Exceptions
Definem exceções personalizadas para tratar erros específicos da aplicação de forma clara e controlada.

🔐 Security
Abrange classes relacionadas à autenticação e autorização, garantindo segurança na aplicação.

📚 Shared
Contêm constantes e utilitários compartilhados entre diferentes partes do sistema.

### Features

| **Feature**                   | **Descrição**                                          |
|-------------------------------|-------------------------------------------------------|
| **Static Site Generation**    | Utilização de Flyway para gerenciamento de migrações de banco de dados, permitindo configurações consistentes e alto desempenho. |
| **Nested Pages**              | Estrutura hierárquica nos serviços e controladores para modularidade e organização de endpoints. |
| **Interactive Code Blocks**   | Código altamente interativo com destaque para uso de `Spring Boot`, `Spring Security` e bibliotecas como `Firebase` e `MinIO`. |
| **Custom Markdown Components**| Criação de componentes reutilizáveis nos mappers para conversão eficiente de DTOs e entidades. |
| **SEO-Optimized**             | Uso de ferramentas como o SpringDoc OpenAPI para documentar e expor APIs de forma eficiente. |
| **Syntax Highlighting**       | Utilização de logs otimizados com Loki para maior legibilidade e rastreabilidade. |
| **Pagination**                | Paginação eficiente implementada em endpoints com integração ao Spring Data JPA. |

### Tecnologias

O backend do projeto Raízes Cearenses utiliza tecnologias modernas para garantir desempenho, segurança e flexibilidade:

- Spring Boot 3.3.4 - Framework robusto para construção de aplicações Java escaláveis e otimizadas para produção.
- Flyway - Ferramenta para gerenciamento e versionamento de migrações de banco de dados, garantindo consistência e controle de alterações no PostgreSQL.
- Spring Security - Solução abrangente para autenticação e controle de acesso, promovendo segurança avançada na aplicação.
- MinIO - Serviço de armazenamento em bucket utilizado para gerenciar imagens de forma eficiente e escalável.
- Freemarker - Template engine poderosa para geração dinâmica de e-mails e relatórios.
- SpringDoc OpenAPI - Documentação interativa e auto-gerada das APIs, facilitando a integração e compreensão dos serviços.
- Loki & Micrometer - Ferramentas para monitoramento e logging eficientes, integrados com Prometheus para métricas detalhadas.
- Firebase Admin SDK - Integração para notificações push e autenticação segura.
